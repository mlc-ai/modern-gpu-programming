Appendix
========


The main tutorial path ends with Writing TIRX Kernels with Agents. The
appendix is not another reading path; use it when you need setup help,
extra practice, debugging checklists, or API details.

How to Use This Appendix
------------------------

+--------------------------------------+-------------------------------+
| Need                                 | Use                           |
+======================================+===============================+
| Install TVM/TIRX or run a tiny       | **Environment Setup**         |
| kernel                               |                               |
+--------------------------------------+-------------------------------+
| Practice basic thread indexing,      | **Practice Kernel: Fused GELU |
| elementwise code, and reductions     | Gate** and **Practice Kernel: |
|                                      | RMSNorm Reduction**           |
+--------------------------------------+-------------------------------+
| Inspect generated CUDA for scope     | **TIRX Language and Compile   |
| guards, barriers, and lowered        | Pipeline**                    |
| instructions                         |                               |
+--------------------------------------+-------------------------------+
| Read the complete Flash Attention 4  | **Flash Attention 4 TIRX      |
| TIRX source                          | Source**                      |
+--------------------------------------+-------------------------------+
| Look up the exact spelling of a TIRX | **TIRX API Lookup**           |
| API used in the tutorial             |                               |
+--------------------------------------+-------------------------------+
| Understand the parser, buffer model, | **TIRX Language and Compile   |
| scopes, metaprogramming, or compile  | Pipeline**                    |
| pipeline                             |                               |
+--------------------------------------+-------------------------------+

Most readers only need **Environment Setup** and **TIRX API Lookup** at
first. The practice kernels are optional.
